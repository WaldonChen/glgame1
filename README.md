glgame1
=======

Little game written in OpenGL and wxWidgets.

Prerequisites
----------------

- wxWidgets 3.0+
- OpenGL
- DevIL
- CMake

Build
-------

```
git clone https://github.com/WaldonChen/glgame1.git
cd glgame1
mkdir build
cd build
cmake ..
make
```

Installation
-------------

```
make install
```
